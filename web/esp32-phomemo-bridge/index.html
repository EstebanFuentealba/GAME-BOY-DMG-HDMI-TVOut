Firmwares

Instala el firmware del Game Boy HDMI TVOut y del puente Bluetooth para imprimir capturas en una Phomemo T02.

RP2040 HDMI TVOut

Conecta el RP2040 manteniendo presionado BOOTSEL. Cuando aparezca la unidad RPI-RP2, usa el instalador UF2.

Descargar UF2

  • Funciona en Chrome o Edge con soporte File System Access.
  • Si el navegador no permite escribir la unidad, descarga el UF2 y copialo manualmente a RPI-RP2.

ESP32-C3 Phomemo Bridge

Conecta el ESP32-C3 por USB, ponlo en modo bootloader si tu placa lo requiere y presiona instalar.

Este navegador no soporta Web Serial. Usa Chrome o Edge en escritorio.
  • Chip esperado: ESP32-C3.
  • La página debe abrirse desde GitHub Pages o un origen HTTPS.
  • El manifest y los binarios se publican automáticamente desde GitHub Actions.
